# **ScamShielder Bot 🛡️🐾**

**ScamShielder** 是一款用於 Discord 的防詐騙連結機器人，能夠自動偵測和刪除可疑連結，保護你的伺服器成員誤觸連結！

---

## **功能 Features**
- 🚫 **自動偵測詐騙連結**：識別常見詐騙網站並自動刪除。  
- 🔇 **禁言違規使用者**：對發送詐騙連結的成員進行暫時禁言。  
- 🛡️ **靈活配置**：可自訂詐騙連結清單，增加伺服器的防護強度。

## 注意事項

為確保本機器人能有效監控並管理詐騙連結，請將 機器人 添加至需要監控的頻道，並賦予以下權限：
- 檢視頻道：允許機器人看到頻道中的訊息。
- 送出訊息：允許機器人送出警告訊息。
- 管理訊息：讓機器人能夠刪除違規訊息。
